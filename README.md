# Attendance Odoo Integration

Python tool to integrate ZKTeco attendance devices with Odoo ERP using REST API.

## Features
- Connect to ZKTeco attendance device
- Extract attendance logs
- Send logs to Odoo ERP via API
- Automate employee attendance synchronization

## Technologies
- Python
- ZKTeco SDK
- REST API
- Odoo ERP

## Use Case
Designed for multi-branch retail environments where attendance devices must automatically sync logs with Odoo ERP.
